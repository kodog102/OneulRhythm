## ✨ Why

많은 생산성 앱은 해야 할 일을 더 많이 보여주고,
더 많은 기능을 제공합니다.

하지만 실제로 중요한 것은
해야 할 일의 개수가 아니라,

**지금 가장 먼저 해야 하는 하나.**라고 생각했습니다.

OneulRhythm은

사용자가 하나의 리듬에만 집중하도록 돕기 위해 만든,

iOS 프로젝트입니다.
