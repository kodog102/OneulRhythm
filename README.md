🌿 OneulRhythm

오늘을 차분하게 이어가는 루틴 앱

OneulRhythm(오늘리듬)은 사용자가 하루를 조금 더 편안하고 의미 있게 보낼 수 있도록 돕는 iOS 라이프스타일 루틴 앱입니다.

복잡한 생산성 앱이 아니라, 오늘 해야 할 리듬을 자연스럽게 이어갈 수 있는 경험을 목표로 합니다.

⸻

✨ Vision

오늘리듬은 다음과 같은 경험을 제공합니다.

* 지금 해야 할 일에 집중하기
* 하루의 흐름을 자연스럽게 이어가기
* 부담 없는 루틴 관리
* 따뜻하고 차분한 사용자 경험

⸻
